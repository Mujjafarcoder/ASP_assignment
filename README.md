# ASP Assignment - Scan Dashboard

This is a React-based security scan dashboard project.

## Features
- Login page
- Dashboard with scan progress
- Scan details page
- Vulnerability status (Critical, High, Medium, Low)
- Dark / Light mode toggle

## Technologies Used
- React.js
- Bootstrap
- FontAwesome
- JSON Server (Mock API)

## Installation

Clone the repository:

git clone https://github.com/Mujjafarcoder/ASP_assignment.git

Install dependencies:

npm install

Run the project:

npm start

## Deployment

The project is deployed using Vercel.
